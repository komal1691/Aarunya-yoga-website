# Aarunya Yoga & Meditation Website

A multi-page wellness website created to provide structured yoga and meditation learning resources.

## Features
- Categorized yoga and meditation sections
- Embedded guided videos
- Clean and responsive UI
- Informational About and Contact pages

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This project was developed as a UI-focused academic website to design a simple platform for yoga and meditation learners.
